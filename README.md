## Repository Structure

- `src/` core Python modules (OO calculators, models, utilities)
- `config/` default and example config files (YAML/JSON)
- `data_output/` generated results (gitignored if desired)
- `env/` environment files (e.g., `environment.yml`, `.env.example`)
- `tests/` unit and integration tests
- `docs/` design notes and references
